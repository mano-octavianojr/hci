Coursework

  • Programming (30%)
  • Project (70%)

Programming

You will be required to complete three programming assignments, each contributing 10% to your final grade . These assignments are designed to reinforce the concepts and skills you've learned throughout the course:

  • Programming #1:: SQL using Python
  • Programming #2:: Data Mining using Python
  • Programming #3:: ML Application using Python

Important: All code must be uploaded to a GitHub repository. Your GitHub repositories will be checked for completeness and correctness, so make sure your code is properly organized and documented. If you are not yet familiar with GitHub, please take the time to learn how to:

  • Create a GitHub account.
  • Create and manage repositories.
  • Commit and push code.
  • Share repository links.

Project

Students will work in pairs to propose a goal they intend to achieve using the provided dataset. Their objective is to analyze the dataset based on the goal they have in mind. Students may use various tools to examine the data and produce a meaningful analysis.

Schedule of presentation and grading criteria will be announced in class.

Deliverables: A 12 - 15 slide presentation that provides a short yet insightful analysis of the dataset.
